

# إضافة زر "فاتورة ضريبية" في صفحة تسوية الشركات الرئيسية

## المشكلة
زر "فاتورة ضريبية" موجود حاليا فقط في صفحة تفاصيل الشركة (`/reports/company-settlement/:companyId`). المستخدم يحتاج الوصول إليه من الصفحة الرئيسية (`/reports/company-settlement`).

## الحل
إضافة عمود "إجراءات" في جدول الشركات يحتوي على زر صغير لإنشاء فاتورة ضريبية لكل شركة. عند الضغط عليه يظهر Popover صغير لإدخال نسبة المربح (الافتراضي 10%) ثم زر "إنشاء".

## التعديلات

### ملف: `src/pages/CompanySettlement.tsx`

1. **إضافة عمود "إجراءات"** في جدول الشركات (بجانب "المستحق للشركة")
2. **إضافة زر فاتورة ضريبية** (أيقونة Receipt) في كل صف شركة
3. **عند الضغط**: يظهر Popover يحتوي على:
   - حقل نسبة المربح (رقمي، افتراضي 10%)
   - زر "إنشاء فاتورة"
4. **عند الإنشاء**: يستدعي Edge Function `generate-tax-invoice` مع فلاتر الشهر والنوع والملغية الحالية + company_id من الصف + نسبة المربح
5. يفتح النتيجة في تبويب جديد

### التفاصيل التقنية

- إضافة imports: `Receipt, Loader2` من lucide + `Popover` من UI
- إضافة states: `taxInvoiceCompanyId`, `profitPercent`, `generatingTaxInvoice`
- إضافة دالة `handleGenerateTaxInvoice(companyId)` تستدعي Edge Function
- منع propagation عند الضغط على زر الفاتورة (لمنع التنقل لصفحة التفاصيل)
- إضافة `TableHead` جديد "إجراءات" و `TableCell` في كل صف
