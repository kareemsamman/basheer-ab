

# خطة: نقل شارة التجديدات إلى بجانب "تقارير الوثائق"

## المطلوب
بدلاً من عنصرين منفصلين:
```
├── تقارير الوثائق
├── التجديدات [12]
```

نريد عنصر واحد مع النص والشارة:
```
├── تقارير الوثائق | التجديدات [12]
```

---

## التغييرات التقنية

### تحديث `Sidebar.tsx`

#### 1. إزالة عنصر "التجديدات" المنفصل
```typescript
// السطر 107 - حذف هذا السطر
{ name: "التجديدات", href: "/reports/policies?tab=renewals", icon: RefreshCw, badge: 'renewals' },
```

#### 2. إضافة الشارة إلى "تقارير الوثائق"
```typescript
// السطر 106
{ name: "تقارير الوثائق", href: "/reports/policies", icon: BarChart3, badge: 'renewals' },
```

#### 3. تحديث عرض العنصر ليظهر "التجديدات" مع الشارة
في دالة العرض، نضيف نص "التجديدات" قبل الشارة للعناصر التي تحتوي على `badge: 'renewals'`:

```typescript
// في جزء عرض NavLink
<NavLink ...>
  <item.icon className="..." />
  <span>{item.name}</span>
  {/* إضافة نص التجديدات للشارة الخاصة */}
  {item.badge === 'renewals' && (
    <span className="text-xs text-muted-foreground mr-2">| التجديدات</span>
  )}
  {renderBadge(item)}
</NavLink>
```

## النتيجة المتوقعة

```
التقارير
├── تقارير الوثائق | التجديدات [12]  ← شارة برتقالية مع النص
├── تقرير الشركات (admin)
└── التقارير المالية (admin)
```

1. عنصر واحد فقط "تقارير الوثائق"
2. بجانبه نص "التجديدات" مع الشارة البرتقالية
3. النقر على العنصر يفتح `/reports/policies`
4. العدد يتحدث تلقائياً
