
# فاتورة ضريبية - تقرير HTML جديد لتسوية الشركة

## الفكرة
إنشاء تقرير HTML جديد بعنوان "فاتورة ضريبية" يعمل من صفحة `/reports/company-settlement/:companyId`. يتضمن جدول بتفاصيل البوالص (بدون إلزامي) مع نسبة ربح وهمية يحددها الأدمن.

## كيف يعمل

1. **زر جديد** في صفحة `CompanySettlementDetail` بجانب زر "تقرير PDF" الحالي → "فاتورة ضريبية"
2. عند الضغط يظهر **حقل نسبة مئوية** (مثلا 10%) يكتبها الأدمن
3. يتم إرسال الطلب إلى Edge Function جديدة `generate-tax-invoice`
4. التقرير يرفع إلى Bunny CDN ويفتح في تبويب جديد

## محتوى الفاتورة الضريبية

| # | العميل | الهاتف | رقم الهوية | نوع التأمين | المبلغ الكامل | المربح |
|---|--------|--------|------------|-------------|--------------|--------|
| 1 | أحمد | 050... | 12345 | اراضي مقدسة - ثالث + اكس - خدمات طريق | 1,500 | 150 |

**القواعد:**
- **استبعاد الإلزامي** تماما من الجدول والحسابات
- **للباقات**: تجميع كل بوالص الباقة (بنفس group_id) في سطر واحد، مع عرض تفاصيل كل مكون (شركة + نوع)، والمبلغ = مجموع أسعار المكونات (بدون إلزامي)
- **للبوالص المفردة** (بدون group_id): سطر واحد عادي
- **نوع التأمين**: يعرض اسم الشركة + نوع التأمين لكل مكون
- **المربح**: يحسب كـ (المبلغ الكامل × النسبة المئوية / 100) - وهمي وليس الربح الحقيقي
- **الإجمالي**: مجموع المبالغ ومجموع المربح في أسفل الجدول
- يعمل مع الفلاتر الحالية (تاريخ، نوع، ملغية)

## التفاصيل التقنية

### ملف جديد: `supabase/functions/generate-tax-invoice/index.ts`

**المدخلات:**
```typescript
{
  company_id: string;
  start_date?: string;
  end_date?: string;
  policy_type?: string;
  include_cancelled?: boolean;
  profit_percent: number; // النسبة المئوية (مثلا 10)
}
```

**الاستعلام:**
- جلب بوالص الشركة مع بيانات العميل (full_name, phone_number, id_number) والسيارة
- استبعاد ELZAMI من النتائج
- جلب بوالص نفس group_id من شركات أخرى لعرض تفاصيل الباقة

**منطق التجميع:**
1. تجميع البوالص حسب group_id (إن وجد)
2. لكل مجموعة: جلب كل بوالص الباقة (من كل الشركات، بدون إلزامي) لعرض التفاصيل
3. المبلغ الكامل = مجموع insurance_price للمكونات (بدون إلزامي)
4. المربح = المبلغ الكامل × (profit_percent / 100)

**HTML Template:**
- نفس تصميم AB Insurance (header أزرق، خط Tajawal، RTL)
- عنوان: "فاتورة ضريبية"
- اسم الشركة والفترة
- الجدول: #، العميل، الهاتف، رقم الهوية، نوع التأمين، المبلغ الكامل، المربح
- صف الإجمالي: عدد البوالص، إجمالي المبلغ، إجمالي المربح
- زر طباعة

### ملف: `src/pages/CompanySettlementDetail.tsx`

**إضافات:**
- State جديد: `profitPercent` (number, default 10) و `showTaxInvoiceInput` (boolean)
- زر "فاتورة ضريبية" بجانب زر التقرير الحالي
- عند الضغط: يظهر حقل إدخال النسبة المئوية + زر "إنشاء"
- عند الإنشاء: يستدعي `generate-tax-invoice` مع الفلاتر الحالية + النسبة

### ملف: `supabase/config.toml`
- لا يتم تعديله (يدار تلقائيا)
